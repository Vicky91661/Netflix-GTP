# NETFLIX-GPT
### Create Vite project
    npm create vite@latest
### Integrated TailwindCSS to this Project 
    npm install -D tailwindcss postcss autoprefixer
    npx tailwindcss init -p

- Login / Sign Up
    - Sign In / Sign Up From
    - Redirect to the Browse Page
- Browse Page
    - Header
    - Main Movie
        - Trailer in Background
        - Title & Description
    - Movie Suggestions
        - Movie List
- GPT 
    - Search Bar
